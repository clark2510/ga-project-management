# GA项目管理系统

一个基于HTML、TailwindCSS和JavaScript构建的轻量级项目管理系统，支持项目管理、任务看板、甘特图和数据统计功能。

## 功能特点
- 📋 项目创建与管理
- 🔄 任务看板视图
- 📊 数据统计与可视化
- 📅 甘特图时间轴展示
- 💾 本地存储数据持久化
- 📱 响应式设计

## 在线预览
https://[你的GitHub用户名].github.io/ga-project-management/

## 本地运行
1. 克隆仓库
2. 直接打开 index.html 文件即可运行
